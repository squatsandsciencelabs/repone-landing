RepOne 3D Sensor Troubleshooting

Android

Make sure Bluetooth is on. Do not pair RepOne Device through Settings! ‘Forget’ the device if you have. If on android os 6.0+, make sure Location Services is on. Then,
  1. Force kill RepOne Application
  2. Turn RepOne Device off
  3. Clear Phone Bluetooth Cache
  4. Clear RepOne Application Cache
  5. Restart Phone
  6. Turn Device on
  7. Start RepOne Application
  8. Select a RepOne Device from the list

iOS

Make sure Bluetooth is on. Then,
  1. Force kill RepOne Application
  2. Turn RepOne Device off
  3. Restart Phone
  4. Turn RepOne Device on
  5. Start RepOne Application
  6. Select a RepOne Device from the list

If the above steps do not work, try uninstalling and re-installing the RepOne app on your phone.
Last but not least, if you still have issues make sure to contact us with any issue you may still have. Be sure to include your device brand and model, operating system and operating system version.
